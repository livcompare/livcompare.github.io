# LIV Compare

**A minimal image and video comparison tool for macOS.**

Load up to 4 images or videos at once and compare them with synchronized playback. Built for comparing grades, edits, codecs, upscaling algorithms, or anything else side by side.

[**Download**](https://github.com/livcompare/livcompare.github.io/releases/latest) · [**Homepage**](https://livcompare.github.io)
